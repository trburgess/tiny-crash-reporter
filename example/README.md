-   Install composer

    <http://getcomposer.org/doc/01-basic-usage.md>
-   Install `tiny-crash-reporter` using composer
    ```shell
    composer install
    ```
-   Run the `error-example` script
    ```shell
    php error-example.php
    ```
-   Run the `exception-example` script
    ```shell
    php exception-example.php
    ```
